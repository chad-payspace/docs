## Technical Implementation Examples

### **Quick Integration Code Samples**

<Tabs>
  <Tab title="Accept Payment">
    ```javascript
    // Accept a customer payment
    const payment = await fetch('/api/v2/payment/', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer ' + apiToken,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        amount: 29.99,
        email: 'customer@example.com',
        first_name: 'John',
        last_name: 'Doe',
        ntf_url: 'https://yoursite.com/webhooks/payment'
      })
    });
    ```
  </Tab>
  <Tab title="Process Payout">
    ```javascript
    // Send payout to vendor
    const payout = await fetch('/api/v2/payout/', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer ' + apiToken,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        amount: 250.00,
        email: 'vendor@example.com',
        first_name: 'Jane',
        last_name: 'Smith',
        ntf_url: 'https://yoursite.com/webhooks/payout'
      })
    });
    ```
  </Tab>
  <Tab title="Handle Webhook">
    ```javascript
    // Webhook handler for payment updates
    app.post('/webhooks/payment', (req, res) => {
      const { txid, status, amount } = req.body;
      
      if (status === 'completed') {
        // Payment successful - fulfill order
        fulfillOrder(txid, amount);
      } else if (status === 'failed') {
        // Payment failed - notify customer
        notifyPaymentFailure(txid);
      }
      
      res.status(200).send('OK');
    });
    ```
  </Tab>
</Tabs>
